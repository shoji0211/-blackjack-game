# ブラックジャックゲーム

トランプゲーム「ブラックジャック」のPython実装です。2つのバージョンが利用できます：

- **blackjack.py**: コマンドラインゲーム（ターミナルで実行）
- **app.py**: Webブラウザ版（Flask）

## ゲームルール

- **目的**: ディーラーと対戦し、21に近いカードの合計値を目指す
- **カードの値**:
  - 数字（2-10）: そのままの値
  - J, Q, K: 10
  - A（エース）: 11 または 1（自動で最適な値が選ばれます）

- **ゲームの流れ**:
  1. ベット額を決定
  2. プレイヤーとディーラーに各2枚のカードを配布
  3. プレイヤーのターン：ヒット（カード追加）またはスタンド（終了）を選択
  4. ディーラーのターン：17以上になるまで自動でカードを引く
  5. 結果判定

- **勝利条件**:
  - プレイヤーの合計値がディーラーより21に近い
  - ディーラーがバスト（21超過）でプレイヤーがバストでない
  - プレイヤーが21超過でなければ勝ち

- **特殊**:
  - 21超過 = バスト（負け）
  - 最初の2枚で21 = ブラックジャック（勝ち）

  ## 実行方法

  ### コマンドラインゲーム版

  ```bash
  cd /Users/shojioomori/ブラックジャックゲーム
  python3 blackjack.py
  ```

  ### Webブラウザ版（推奨）

  **1. Flaskをインストール**

  ```bash
  pip3 install -r requirements.txt
  ```

  **2. サーバーを起動**

  ```bash
  cd /Users/shojioomori/ブラックジャックゲーム
  python3 app.py
  ```

  **3. ブラウザでアクセス**

  ```
  http://localhost:5000
  ```

  **4. ゲーム終了**

  ターミナルで `Ctrl + C` を押してサーバーを停止

  ## ゲーム中の操作

  - **ベット額入力**: 現在の残高の範囲内で金額を入力
  - **H キー**: ヒット（カードを1枚追加）
  - **S キー**: スタンド（ターンを終了）
  - **Y/N**: 次のラウンドをプレイするか決定

  ## 初期残高

  初期残高は $100 です。
  すべて失うとゲーム終了となります。

  ## ゲーム例

  ```
  ブラックジャックへようこそ！

  現在の残高: $100
  ベット額を入力してください（1-100）: $20

  【ディーラーの手札】
  手札: 【非表示】, 5♥ (値: 5)

  【プレイヤーの手札】
  手札: K♠, 7♦ (値: 17)

  [H] ヒット, [S] スタンド: S

  【ディーラーの手札】
  手札: Q♣, 5♥ (値: 15)

  ディーラーはヒットします...
  【ディーラーの手札】
  手札: Q♣, 5♥, 8♠ (値: 23)

  ==================================================
  【結果】
  プレイヤー: 17
  ディーラー: 23
  ==================================================

  ディーラーがバスト！プレイヤーの勝ち！
  現在の残高: $140
  ```

  ## コード構成

  - **Card**: カードを表現するクラス
  - **Suit**: スーツ（♥♦♣♠）の列挙型
  - **Deck**: カードデッキを管理するクラス
  - **Hand**: プレイヤーやディーラーの手札を管理するクラス
  - **BlackjackGame**: ゲーム全体を管理するクラス

  ## 機能

  - ✅ プレイヤー vs ディーラーの対戦
  - ✅ ベットシステム
  - ✅ エースの自動値調整
  - ✅ バスト判定
  - ✅ ブラックジャック判定
  - ✅ 複数ラウンドプレイ
  - ✅ 残高管理
